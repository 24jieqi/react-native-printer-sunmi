# react-native-printer-sunmi

printer module for sunmi

## Installation

```sh
npm install react-native-printer-sunmi
```

## Usage

```js
import PrinterSunmi from "react-native-printer-sunmi";

// ...

const result = await PrinterSunmi.multiply(3, 7);
```
